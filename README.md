# ARZ AI

**ARZ - Afet Raporlama ve Zamanlama** için hazırlanmış basit web tabanlı yapay zeka sohbet demosudur.

Bu proje, Google Gemini API kullanarak afet, sağlık, lojistik ve saha yönetimi konularında yapay zeka destekli cevap üretmeyi amaçlar.
